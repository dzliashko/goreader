# Задание 2 — Валидация и свои ошибки
**1.** В ```models/models.go``` создай тип ```ValidationError``` с полями ```Field string``` и ```Message string```, и реализуй на нём метод ```Error() string```.

**2.** Напиши функцию ```(f Feed) Validate() error``` — метод на ```Feed```, который проверяет:

 - ```Title``` не пустой
 - ```URL``` не пустой
 - ```URL``` начинается с ```"http"``` (используй ```strings.HasPrefix```)

При первой же ошибке возвращай ```&ValidationError{...}```. Если всё хорошо — ```nil```.

**3.** Измени ```NewFeed``` — пусть теперь она возвращает ```(Feed, error)``` и внутри вызывает ```Validate()```.

**4.** В ```main.go``` обнови вызов ```NewFeed``` под новую сигнатуру. Добавь два вызова: один с валидными данными, один с намеренно сломанным URL (например, пустая строка) — и обработай оба через ```if err != nil```.

Подсказка: ```strings.HasPrefix(s, prefix string) bool``` — нужен импорт ```"strings"```.
