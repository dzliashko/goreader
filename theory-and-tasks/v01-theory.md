##Методы на структурах
В Go можно "привязать" функцию к типу. Такая функция называется методом:
```
// Обычная функция:
func PrintFeed(f Feed) {
    fmt.Println(f.Title)
}

// Метод на структуре Feed:
func (f Feed) Print() {
    fmt.Println(f.Title)
}

// Вызов:
feed.Print()
```
Часть (f Feed) называется получателем (receiver). f — это копия Feed, на которой вызван метод.

## Указатель-получатель
Если метод должен изменить структуру, нужен указатель *:
```
func (f *Feed) SetTitle(title string) {
    f.Title = title  // меняем оригинал, не копию
}
```
